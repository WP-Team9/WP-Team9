# WP-Team9

실행 환경 설정

### `cd wp-team9`

### `npm install`

### `npm install axios`

### `npm install react-router-dom`

### `npm install @material-ui/core --legacy-peer-deps`

### `npm install @mui/icons-material --legacy-peer-deps`

### `npm install react-slick --save`

### `npm install slick-carousel --save`

### `npm start`
